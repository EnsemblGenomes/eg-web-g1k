Features

The 1000Genomes browser adds these enhancements to the standard Ensembl Human genome browser

Browser configuration options

  • 1000 Genomes menu in the configuration of the region view
    • Phase 3 integrated release - 1000 Genomes variation
  • Display of variation tracks: you can choose to display the density of the variations

Browser interface

  • Zoom-in display panel in the Variation view

Tools

  • Get VCF data from the 1000 Genomes VCF file for the current view. On Location, Gene, Variation, Transcript and other views, use the tool button located in the left-hand menu:

    Get VCF data

  • Additional tools
    • Data Slicer
    • Variation Pattern Finder
    • VCF to PED converter
