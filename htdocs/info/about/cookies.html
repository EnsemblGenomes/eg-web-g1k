List of the cookies set by the genome browser and what they are used for

ENSEMBL_WIDTH

used to configure the width of the ensembl images

DYNAMIC_WIDTH

remembers the best fit value for the ensembl image width

ENSEMBL_AJAX

if set allows the asynchronous  load of the ensembl web pages

ENSEMBL_HINTS

if set allows the display of the hints on the ensembl pages

ENSEMBL_IMAGE_WIDTH

user configured width of the ensembl images

ENSEMBL_WWW_USER

remembers the user if logged in

ENSEMBL_WWW_SESSION

remembers the session settings, e.g switched on tracks
